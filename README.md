# Poseidon Landing

Многостраничный лендинг на `React + TypeScript + Vite + Tailwind CSS`.

## Страницы

- `/` — главная
- `/services` — услуги
- `/cases` — кейсы
- `/about` — о компании
- `/contacts` — контакты
- `/private-clients`, `/business-clients` — лендинги по сегментам

## Запуск

```bash
npm install
npm run dev
```

## Проверки

```bash
npm run lint
npm run build
```
